### step 3 异常处理
- 新增自定义异常处理
#### 异常处理程序知识
- [参考](https://www.runoob.com/cplusplus/cpp-exceptions-handling.html)

#### 异常处理设计
- 几种异常

  2. 输入文件异常（输入曲线不符合格式，输入线段数目，“乱码”输入）
  3. 直线异常
  4. 曲线异常
